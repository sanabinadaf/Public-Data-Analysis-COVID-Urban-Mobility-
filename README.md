# Public-Data-Analysis-COVID-Urban-Mobility-

Analysis of public data related to COVID-19 and urban mobility patterns
 Collected and cleaned large public datasets related to COVID-19, environmental conditions, and urban mobility
 Performed time-series analysis to identify trends, seasonal patterns, and sudden spikes
 Conducted geospatial analysis to detect regional hotspots and mobility patterns
 Built predictive models to forecast future case growth and pollution levels
 Designed interactive Power BI dashboards and visualizations to present insights clearly
 Provided data-driven insights to support public safety measures and urban planning
